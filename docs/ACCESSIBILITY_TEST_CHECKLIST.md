# Voltune accessibility checklist

Automated tests cover basic labels and interaction targets. The items below require a person and remain unchecked until performed.

- [ ] TalkBack reads controls in a useful order on library, settings, mini-player, and full player.
- [ ] Play/pause, repeat, shuffle, favorite, timer, equalizer, and volume-normalization states are announced.
- [ ] Switch Access can reach every command without trapping focus.
- [ ] A hardware keyboard can navigate, activate, and dismiss controls.
- [ ] Text at 150%, 180%, and 200% does not overlap or disappear.
- [ ] Long Russian and English track, artist, album, genre, and playlist names remain understandable.
- [ ] Rotation and Activity recreation preserve the current UI and playback state.
- [ ] Split-screen keeps primary controls reachable.
- [ ] Custom themes retain readable contrast.
- [ ] State is never communicated by color alone.
- [ ] Particles can be disabled independently.
- [ ] Cover rotation can be disabled.
- [ ] Reduced-motion mode removes nonessential movement without breaking gestures or navigation.
- [ ] Dialogs can be dismissed with Back and the documented horizontal swipe.

When custom text has insufficient contrast, Voltune enables a contrasting outline while preserving the selected text color and explains this in theme settings.
