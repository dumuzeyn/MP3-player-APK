# MP3 Player 2.4.2

## Русский

Официальное обновление локального MP3-плеера для Android.

### Главное

- Повтор песни и повтор списка продолжают воспроизведение в фоне, пока пользователь не нажмёт паузу, не сработает таймер сна или воспроизведение не будет действительно прервано.
- Сервис сохраняет очередь и позицию и восстанавливает проигрывание после временного отказа аудиофокуса.
- Удаление песни из библиотеки или очереди больше не рассинхронизирует мини-плеер, большой плеер и системное уведомление.
- Импорт папок, обновление метаданных и чтение обложки для уведомления выполняются без блокировки интерфейса.
- Прозрачность отдельно настраивается для песен, избранного, плейлистов, жанров, исполнителей, альбомов, настроек, мини-плеера, шапки и диалогов.
- Снижена нагрузка длинных списков, вкладок и анимированных карточек плейлистов.
- Исправлены кнопка GitHub, выравнивание времени в большом плеере и выравнивание громкости на Android 8.

Фоновый повтор списка с таймером сна и удалением задачи проверен автоматическим тестом на физическом realme с Android 13.

## English

Official update of the local MP3 player for Android.

### Highlights

- Repeat-one and repeat-all continue in the background until the user pauses playback, the sleep timer expires, or playback is genuinely interrupted.
- The service preserves the queue and position and recovers after temporary audio-focus denial.
- Removing a song from the library or queue no longer desynchronizes the mini-player, full player, and system media notification.
- Folder import, metadata refresh, and notification artwork decoding no longer block the interface.
- Opacity can be adjusted independently for songs, favorites, playlists, genres, artists, albums, settings, mini-player, header, and dialogs.
- Long lists, tabs, and animated playlist cards use less memory and background work.
- Fixed the GitHub button, full-player time alignment, and Android 8 volume-leveling fallback.

Background repeat with a sleep timer and removed app task was verified by an automated test on a physical realme device running Android 13.
