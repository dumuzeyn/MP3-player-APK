# Security Policy

## Supported version

Security fixes are applied to the latest published release of MP3 Player Voltune.

## Reporting a vulnerability

Do not open a public issue for a vulnerability. Use [GitHub private vulnerability reporting](https://github.com/dumuzeyn/MP3-Player-Voltune/security/advisories/new) and include:

- the affected app version and Android version;
- reproducible steps and expected impact;
- sanitized logs without music URIs, file paths, passwords, or signing material.

The report will be reviewed before public disclosure. Never commit release keystores, passwords, or user music data to the repository.
